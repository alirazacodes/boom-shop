# Boom Market Smart Contract

## Installation

1. Clone the repository:

```bash
git clone https://github.com/alirazacodes/boom-shop.git
```

2. Install dependencies:

```bash
npm install
```


3. Add further contracts with:

```bash
clarinet contract new <contract-name>
```

4. Run the tests with:

```bash
npm test
```

5. Check Smart contract:

```bash
clarinet check
```
